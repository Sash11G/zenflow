I learned isinstance()

isinstance() is a built-in Python function that checks
if an object is an instance of a specific class or a tuple
of classes. 
Returns: True or False

isinstance(5, int)              # True
isinstance("hello", str)        # True
isinstance([1, 2, 3], list)     # True
isinstance(5, (int, float))     # True
isinstance(5.0, (int, float))   # True
isinstance("5", (int, float))   # False
